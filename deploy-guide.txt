# Running Crypto Predictor PRO 24/7 — Free Cloud Deployment

## Best option: Oracle Cloud (Always Free — $0 forever)

Oracle gives you a real server that never expires and never charges.
It's more setup than Railway but it's completely free and has no limits.

---

## Step 1 — Create Oracle Cloud account

1. Go to https://cloud.oracle.com
2. Click "Start for free"
3. Fill in details — requires a credit card but will NOT be charged
4. Select your home region (pick one close to you — you can't change this later)
5. Wait for account activation (can take up to 30 minutes)

---

## Step 2 — Create a free VM

1. Log into Oracle Cloud Console
2. Click the menu (☰) → Compute → Instances
3. Click "Create instance"
4. Name it: `crypto-predictor`
5. Image: **Ubuntu 22.04** (click "Change image" if needed)
6. Shape: Click "Change shape" → select **VM.Standard.A1.Flex** → set 1 OCPU, 6GB RAM → this is the Always Free shape
7. Under "Add SSH keys" → click "Generate a key pair for me" → Download both keys (save them somewhere safe)
8. Click "Create"

Wait about 2 minutes for it to start. Copy the **Public IP address** shown.

---

## Step 3 — Connect to your server

On Windows, open Command Prompt and run:

```bash
ssh -i path\to\your\private_key.key ubuntu@YOUR_IP_ADDRESS
```

Example:
```bash
ssh -i C:\Users\Anit\Downloads\ssh-key.key ubuntu@152.70.123.45
```

If it asks "Are you sure you want to continue connecting?" type `yes`

---

## Step 4 — Install Node.js and Python on the server

Once connected, run these commands one by one:

```bash
# Update system
sudo apt update && sudo apt upgrade -y

# Install Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# Install Python and pip
sudo apt install -y python3 python3-pip

# Install Python packages
pip3 install xgboost scikit-learn pandas numpy

# Verify
node --version
python3 --version
```

---

## Step 5 — Upload your project files

On your Windows machine, open a NEW Command Prompt (keep the SSH one open) and run:

```bash
scp -i C:\Users\Anit\Downloads\ssh-key.key -r C:\Users\Anit\OneDrive\Desktop\Test\Trader\crypto-predictor ubuntu@YOUR_IP_ADDRESS:~/
```

This copies your entire project folder to the server.

---

## Step 6 — Install Node packages on server

Back in your SSH terminal:

```bash
cd ~/crypto-predictor
npm install
```

---

## Step 7 — Open firewall port 3000

In Oracle Cloud Console:
1. Go to your instance → click the VCN name
2. Security Lists → Default Security List
3. Add Ingress Rule:
   - Source: 0.0.0.0/0
   - Destination Port: 3000
   - Protocol: TCP

Also run on the server:
```bash
sudo iptables -I INPUT -p tcp --dport 3000 -j ACCEPT
sudo netstat-persistent save  # saves rule after reboot
```

---

## Step 8 — Run with PM2 (keeps it alive 24/7)

```bash
# Install PM2 process manager
sudo npm install -g pm2

# Start the server
pm2 start server.js --name crypto-predictor

# Make it auto-start on reboot
pm2 startup
pm2 save
```

Your server is now running at: `http://YOUR_IP_ADDRESS:3000`

---

## Step 9 — Monitor it

```bash
# See live logs
pm2 logs crypto-predictor

# See status
pm2 status

# Restart if needed
pm2 restart crypto-predictor

# Stop
pm2 stop crypto-predictor
```
